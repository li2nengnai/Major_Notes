## 核心概念

### [[Collection]] 体系

```
Collection
├── List（有序、可重复、有索引）
│   ├── ArrayList     ← 数组，查询快，增删慢
│   └── LinkedList    ← 链表，增删快，查询慢
├── Set（不重复、无索引）
│   ├── HashSet       ← 无序
│   ├── LinkedHashSet ← 有序
│   └── TreeSet       ← 可排序（红黑树）
```

### [[Map]] 体系

```
Map（键值对，键不重复）
├── HashMap           ← 无序，允许 null
├── LinkedHashMap     ← 有序
├── TreeMap           ← 键可排序
└── ConcurrentHashMap ← 线程安全
```

### [[ArrayList]] 扩容机制

默认容量 10，扩容 1.5 倍，用 `Arrays.copyOf()` 拷贝到新数组。

### HashMap 底层（JDK8）

**数组 + 链表 + 红黑树**。hash 算法：`(h = key.hashCode()) ^ (h >>> 16)`。链表长度 ≥8 时转红黑树。

## 语法格式

```java
// List
List<String> list = new ArrayList<>();
list.add("A"); list.get(0); list.remove(0); list.size();

// Set
Set<Integer> set = new HashSet<>();   // 无序
Set<Integer> set2 = new TreeSet<>();  // 排序

// Map
Map<String, Integer> map = new HashMap<>();
map.put("key", 1); map.get("key"); map.containsKey("key");
```

### 三种遍历方式

```java
// 1. 迭代器
Iterator<String> it = list.iterator();
while (it.hasNext()) { String s = it.next(); }

// 2. 增强 for
for (String s : list) { }

// 3. forEach + Lambda
list.forEach(s -> System.out.println(s));
```

## 踩坑记录

> [!warning] 迭代器遍历时用 list.remove()
> **现象**：`ConcurrentModificationException`
> ```java
> for (String s : list) {
>     if (条件) list.remove(s);  // ❌ 抛异常
> }
> ```
> **解决**：用迭代器的 `remove()`
> ```java
> Iterator<String> it = list.iterator();
> while (it.hasNext()) {
>     String s = it.next();
>     if (条件) it.remove();  // ✅
> }
> ```

> [!warning] HashMap 键重复时值覆盖
> ```java
> map.put("a", 1);
> map.put("a", 2);
> System.out.println(map.get("a"));  // 2（被覆盖了）
> ```

## 实战案例

```java
// 统计单词出现次数
String text = "a b c a b a";
Map<String, Integer> freq = new HashMap<>();
for (String word : text.split(" ")) {
    freq.put(word, freq.getOrDefault(word, 0) + 1);
}
System.out.println(freq);  // {a=3, b=2, c=1}
```
