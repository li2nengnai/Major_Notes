# 集合框架（Collection / List / Set / Map）

## 📌 课程模块

pk015-pk016 · 集合体系/List/Set/Map/Collections · **高频考点**

## 💻 核心代码示例

```java
// 1. Collection 体系特点
public class CollectionTest1 {
    public static void main(String[] args) {
        // ArrayList：有序、可重复、有索引
        ArrayList<String> list = new ArrayList<>();
        list.add("java1");
        list.add("java2");
        list.add("java1");
        System.out.println(list);                    // [java1, java2, java1]

        // HashSet：无序、不重复、无索引
        HashSet<String> set = new HashSet<>();
        set.add("java1");
        set.add("java2");
        set.add("java1");
        System.out.println(set);                     // 随机顺序，只有1个java1
    }
}
```

```java
// 2. List 特有方法（索引操作）
public class ListTest1 {
    public static void main(String[] args) {
        List<String> list = new ArrayList<>();
        list.add("蜘蛛精");
        list.add("至尊宝");
        list.add("牛夫人");

        list.add(2, "紫霞仙子");            // 指定索引插入

    }
}
```

```java
// 3. 三种遍历方式（迭代器 / 增强for / forEach）
// 方式1：迭代器 Iterator
Collection<String> c = new ArrayList<>();
c.add("赵敏"); c.add("小昭"); c.add("素素");

Iterator<String> it = c.iterator();
while (it.hasNext()) {                     // hasNext() 判断是否有下一个
    String ele = it.next();                // next() 获取元素
    System.out.println(ele);
}

// 方式2：增强 for
for (String ele : c) {
    System.out.println(ele);
}

// 方式3：forEach + Lambda（JDK8+）
c.forEach(s -> System.out.println(s));
```

## 🧩 知识点拆解

### 知识点1：Collection 体系

- **是什么**：集合框架顶层接口，List（有序可重复有索引）、Set（不重复无索引）
- **老师强调的要点**：
  - `ArrayList`：查询快，增删慢（数组结构）
  - `LinkedList`：增删快，查询慢（链表结构）
  - `HashSet`：基于哈希表，无序
  - `LinkedHashSet`：基于哈希表+双向链表，有序
  - `TreeSet`：基于红黑树，可排序

### 知识点2：Map 体系

- **是什么**：键值对存储，键不重复
- **老师强调的要点**：
  - `HashMap`：键无序（允许 null 键/值）
  - `LinkedHashMap`：键有序
  - `TreeMap`：键可排序（键需实现 Comparable）
  - `put()` 键重复时值覆盖，返回被覆盖的旧值

### 知识点3：迭代器遍历（重要！）

- **是什么**：Iterator 是集合的通用遍历方式
- **老师强调的要点**（**易错点！**）：
  - `hasNext()` 判断是否还有下一个元素
  - `next()` 获取下一个元素，同时指针后移
  - 迭代器遍历时**不能用集合自身的 remove()**，必须用迭代器的 `remove()`

## ⚠️ 常见考题 / 易错点

1. **选择题——迭代器遍历时删除元素**：
   ```java
   // ❌ 错误：会抛 ConcurrentModificationException
   for (String s : list) { list.remove(s); }
   // ✅ 正确：使用 Iterator 的 remove()
   Iterator<String> it = list.iterator();
   while (it.hasNext()) { it.next(); it.remove(); }
   ```

2. **填空题——HashMap 底层**：JDK8 后 = **数组 + 链表/红黑树**

3. **选择题——TreeSet 排序要求**：元素必须实现 **Comparable** 接口，或在构造时传入 **Comparator**

4. **选择题——List 与 Set 区别**：
   - List 可重复有索引 ✅
   - Set 不重复无索引 ✅
   - List 不能有 null ❌
   - Set 基于哈希表 ✅

5. **填空题——Map 遍历方式**：
   - `keySet()` 获取所有键，遍历键取值
   - `entrySet()` 获取所有键值对
   - `forEach(k, v)` Lambda 遍历
