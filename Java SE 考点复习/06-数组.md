## 核心概念

数组是**固定长度**的连续内存空间，存储相同类型的数据。

### 两种初始化方式

```java
// 静态初始化：已知元素
int[] arr = {11, 22, 33};
// 完整写法：new int[]{11, 22, 33}

// 动态初始化：只指定长度（系统给默认值）
int[] arr2 = new int[5];  // 默认值 0
```

动态初始化默认值：int→0, double→0.0, boolean→false, 引用→null

### 引用赋值 vs 深拷贝

```java
// ❌ 引用赋值：两个变量指向同一数组
int[] arr3 = arr;
arr3[0] = 999;
System.out.println(arr[0]);  // 999！原数组被改了

// ✅ 深拷贝：创建新数组
int[] copy = new int[arr.length];
for (int i = 0; i < arr.length; i++) {
    copy[i] = arr[i];
}
copy[0] = 999;
System.out.println(arr[0]);  // 11，互不影响
```

## 语法格式

```java
int[] arr = new int[3];     // 推荐写法
int arr[] = new int[3];     // C 语言风格，不推荐

arr.length                  // 数组长度
arr[0]                      // 第一个元素（索引从 0 开始）
arr[arr.length - 1]         // 最后一个元素
```

## 实战案例

### 数组反转（双指针）

```java
int[] arr = {10, 20, 30, 40, 50};
for (int i = 0, j = arr.length - 1; i < j; i++, j--) {
    int temp = arr[j];
    arr[j] = arr[i];
    arr[i] = temp;
}
```

### 评委打分（去最高最低求平均）

```java
public static double getAvg(int[] scores) {
    int sum = 0, max = scores[0], min = scores[0];
    for (int s : scores) {
        sum += s;
        if (s > max) max = s;
        if (s < min) min = s;
    }
    return 1.0 * (sum - min - max) / (scores.length - 2);
}
```

## 踩坑记录

> [!warning] 数组索引越界
> **现象**：`ArrayIndexOutOfBoundsException`
> ```java
> int[] arr = {1, 2, 3};
> System.out.println(arr[3]);  // ❌ 索引最大 2
> ```
> **解决**：确保索引在 `0 ~ arr.length-1` 范围内。

> [!warning] 引用赋值 ≠ 拷贝
> ```java
> int[] a = {1,2,3};
> int[] b = a;     // 不是拷贝！a 和 b 指向同一个数组
> b[0] = 999;
> System.out.println(a[0]);  // 999（被改了）
> ```
