# 一、网络编程（Socket）

## 💻 核心代码示例

```java
// 1. 服务端
import java.net.ServerSocket;
import java.net.Socket;

public class Server {
    private static final int PORT = 8888;            // 端口号

    public static void main(String[] args) {
        try (ServerSocket serverSocket = new ServerSocket(PORT)) {
            System.out.println("服务端已启动，等待客户端连接...");

            // accept() 阻塞等待客户端连接
            Socket clientSocket = serverSocket.accept();
            System.out.println("客户端已连接：" +
                    clientSocket.getRemoteSocketAddress());

            // 多线程：分别处理发送和接收
            Thread receiveThread = new Thread(new ReceiveHandler(clientSocket));
            receiveThread.start();

            Thread sendThread = new Thread(new SendHandler(clientSocket));
            sendThread.start();

            sendThread.join();
            receiveThread.join();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

```java
// 2. 客户端
import java.net.Socket;

public class Client {
    private static final String SERVER_HOST = "localhost";
    private static final int SERVER_PORT = 8888;

    public static void main(String[] args) {
        try (Socket socket = new Socket(SERVER_HOST, SERVER_PORT)) {
            System.out.println("已连接到服务端");

            // 多线程：分别处理发送和接收
            Thread receiveThread = new Thread(new ReceiveHandler(socket));
            receiveThread.start();

            Thread sendThread = new Thread(new SendHandler(socket));
            sendThread.start();

            sendThread.join();
            receiveThread.join();
        } catch (Exception e) {
            e.printStackTrace();
        }
    }
}
```

```java
// 3. 收发处理（核心模式：独立线程）
// 接收消息线程
static class ReceiveHandler implements Runnable {
    private BufferedReader reader;

    public ReceiveHandler(Socket socket) {
        // 获取输入流，读取对方发来的消息
        this.reader = new BufferedReader(
                new InputStreamReader(socket.getInputStream()));
    }

    @Override
    public void run() {
        String message;
        while ((message = reader.readLine()) != null) {
            if ("exit".equalsIgnoreCase(message.trim())) {
                break;                           // 对方要求断开
            }
            System.out.println(message);
        }
    }
}

// 发送消息线程
static class SendHandler implements Runnable {
    private PrintWriter writer;
    private Scanner consoleScanner;

    public SendHandler(Socket socket) {
        // 获取输出流，向对方发送消息
        this.writer = new PrintWriter(socket.getOutputStream(), true);
        this.consoleScanner = new Scanner(System.in);
    }

    @Override
    public void run() {
        String input;
        while (true) {
            input = consoleScanner.nextLine();
            if ("exit".equalsIgnoreCase(input.trim())) {
                writer.println("exit");          // 通知对方断开
                break;
            }
            writer.println(input);               // 发送消息
        }
    }
}
```

## 🧩 知识点拆解

### 知识点1：Socket 通信模型

- **是什么**：Socket 是网络通信的端点，实现两台计算机之间的数据传输
- **工作流程**：
  1. 服务端创建 `ServerSocket`，绑定端口，调用 `accept()` 等待
  2. 客户端创建 `Socket`，连接服务端的 IP 和端口
  3. 连接建立后，通过 `InputStream`/`OutputStream` 收发数据
  4. 通信结束，关闭资源
- **老师强调的要点**：
  - `ServerSocket.accept()` **阻塞等待**直到有客户端连接
  - 客户端和服务端的端口号要一致
  - `localhost` = 本机地址（127.0.0.1）

### 知识点2：多线程处理双向通信

- **是什么**：用两个独立线程分别处理接收和发送，避免阻塞
- **老师强调的要点**：
  - 接收和发送必须分开线程，否则会相互阻塞
  - `BufferedReader` 读取一行，`PrintWriter` 发送一行
  - `exit` 作为断开连接的约定消息

## ⚠️ 常见考题 / 易错点

1. **填空题——Socket 编程流程**：服务端 `ServerSocket` → `accept()` 等待，客户端 `Socket` → 连接
2. **选择题——accept() 作用**：阻塞等待客户端连接，返回 Socket 对象
3. **选择题——多线程原因**：收发分开独立线程，避免 `readLine()` 阻塞导致无法发送

    // 生成8位不重复卡号
    private String createCardId() {
        while (true) {
            String cardId = "";
            Random r = new Random();
            for (int i = 0; i < 8; i++) {
                cardId += r.nextInt(10);          // 数字拼接
            }
            Account acc = getAccountByCardId(cardId);
            if (acc == null) return cardId;       // 无重复则返回
        }
    }

    // 根据卡号查询账户
    private Account getAccountByCardId(String cardId) {
        for (int i = 0; i < accounts.size(); i++) {
            Account acc = accounts.get(i);
            if (acc.getCardId().equals(cardId)) {
                return acc;
            }
        }
        return null;
    }
}

## 🧩 知识点拆解

### 知识点：综合应用

- **涵盖考点**：
  - 面向对象封装：Account 类 JavaBean
  - ArrayList 集合：存储多个账户
  - 流程控制：菜单循环 + switch 分支
  - 字符串比较：`equals()` 比较卡号和密码
  - Random 生成随机数：8位卡号
  - 方法设计：各功能独立封装

## ⚠️ 综合考题方向

1. **对象设计**：Account 类的封装（属性+getter/setter+构造器）
2. **集合应用**：ArrayList 存储多个账户，根据卡号查找
3. **流程控制**：while 循环菜单 + switch 分支 + return 退出
4. **字符串比较**：密码/卡号比较必须用 `equals()` 不能用 `==`
5. **改进建议**：字符串拼接 `cardId += r.nextInt(10)` 推荐改用 `StringBuilder`
