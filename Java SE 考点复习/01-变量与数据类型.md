## 核心概念

### [[变量]]是什么

变量是内存中存储数据的容器。格式：`数据类型 变量名 = 数据;`

- `=` 不是"等于"，是**赋值**，从右向左执行
- `age2 = age2 + 1` 是先取右边 `age2` 的当前值（19），计算 19+1=20，再赋给 `age2`
- 变量可以重复赋值，但**不能改变数据类型**

### 变量的作用域

变量从定义开始到所在 `}` 截止，超出范围无法访问。内部代码块可以访问外部变量，反过来不行。同一作用域内不能定义同名变量。

### [[ASCII]] 编码

字符参与运算时使用对应的 ASCII 值：`'a'=97`、`'A'=65`、`'0'=48`

```java
System.out.println('a' + 10);  // 97 + 10 = 107
// 大小写字母相隔32：'a' - 'A' = 32
```

## 语法格式

### 变量声明

```java
// 数据类型 变量名 = 数据;
int age = 23;
double score = 99.5;
```

### 进制写法（了解即可）

```java
int a1 = 0B01100001;  // 二进制（0B开头）
int a2 = 0141;        // 八进制（0开头，容易混淆，避免使用）
int a3 = 0XFA;        // 十六进制（0X开头）
```

## 实战案例

```java
// 钱包余额变化
double money = 9.5;
money = money + 10;   // 收红包 → 19.5
money = money - 5;    // 发红包 → 14.5
```

## 踩坑记录

> [!warning] 变量未赋值就使用
> **现象**：编译报错
> ```java
> int a;
> System.out.println(a);  // ❌ 编译错误：可能尚未初始化变量
> ```

> [!warning] 变量超出作用域访问
> **现象**：编译报错
> ```java
> { int b = 5; }
> System.out.println(b);  // ❌ 找不到符号
> ```

> [!warning] 类型不匹配赋值
> ```java
> int age = 21;
> age = 35.9;  // ❌ 编译错误：int 不能存 double
> ```
