# [[IO]] 流

## 📌 课程模块

pk018-pk019 · File/字节流/字符流/缓冲流/对象流 · **理解考点**

## 核心概念

### [[字节流]] vs [[字符流]]

| 对比 | 字节流 | 字符流 |
|------|--------|--------|
| 基类 | `InputStream`/`OutputStream` | `Reader`/`Writer` |
| 单位 | 字节 | 字符（2字节） |
| 适用 | **所有文件**（图片/视频/文本） | **纯文本** |
| 汉字 | 可能乱码 | 无乱码 |

### [[缓冲流]]

用**装饰器模式**包装基础流，自带 8KB 缓冲区，减少磁盘 I/O 次数。

```java
new BufferedInputStream(new FileInputStream("a.txt"));
```

### [[try-with-resources]]

JDK7 引入，自动关闭资源。资源必须实现 `AutoCloseable`：

```java
try (InputStream is = new FileInputStream("a.txt")) {
    // 用 is，自动关闭
} catch (IOException e) { }
```

### [[对象序列化]]

Java 对象 → 字节序列存文件。类必须实现 `Serializable`（标记接口），`transient` 修饰的字段不参与序列化。

## 语法格式

```java
// 字节流读文件
InputStream is = new FileInputStream("a.txt");
byte[] all = is.readAllBytes();   // 一次读完（小文件推荐）
System.out.println(new String(all));
is.close();

// 缓冲流 + byte[]（推荐方式）
BufferedInputStream bis = new BufferedInputStream(new FileInputStream("a.txt"));
BufferedOutputStream bos = new BufferedOutputStream(new FileOutputStream("b.txt"));
byte[] buf = new byte[1024];
int len;
while ((len = bis.read(buf)) != -1) {
    bos.write(buf, 0, len);
}

// 字符流（文本无乱码）
try (Reader fr = new FileReader("a.txt")) {
    char[] buffer = new char[1024];
    int len;
    while ((len = fr.read(buffer)) != -1) {
        System.out.print(new String(buffer, 0, len));
    }
}
```

## 踩坑记录

> [!warning] 字节流读汉字会乱码
> **原因**：一个汉字占 3 个字节，一次读一个字节会拆散。
> **解决**：读文本用字符流 `FileReader`，不要用字节流。

> [!warning] 流用完必须关闭
> 不关会导致**文件被占用**或**内存泄漏**。推荐 try-with-resources。

> [!warning] 序列化类必须加 serialVersionUID
> ```java
> public class User implements Serializable {
>     private static final long serialVersionUID = 1L;
>     // 不加的话，类改了字段后反序列化会失败
> }
> ```
