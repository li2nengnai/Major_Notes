# [[流程控制]]（分支 + 循环）

## 📌 课程模块

pk003 · if/switch/for/while/do-while/break/continue/Random · **高频考点**

## 核心概念

### 分支结构

| 结构 | 适用场景 | 特点 |
|------|---------|------|
| `if` | 范围判断（`score >= 60`） | 灵活，支持区间 |
| `switch` | 等值判断（`week.equals("周一")`） | 支持 String，注意 break |

### 三种循环

| 循环 | 特点 | 适用场景 |
|------|------|---------|
| `for` | 先判断后执行，循环变量在内部定义 | 知道次数 |
| `while` | 先判断后执行，循环变量在外部定义 | 不确定次数 |
| `do-while` | **先执行一次**再判断 | 至少执行一次 |

### `break` vs `continue`

- `break`：**跳出**整个循环，循环结束
- `continue`：**跳过**本次，继续下一次

### [[Random]] 公式

生成 `[min, max]` 的随机整数：
```java
int num = r.nextInt(max - min + 1) + min;
// 例：[3, 17] → r.nextInt(15) + 3
```

## 语法格式

### if 三种形式

```java
// 单 if
if (条件) { }

// if-else
if (条件) { } else { }

// if-else if-else
if (条件1) { } else if (条件2) { } else { }
```

### switch（注意 break！）

```java
switch (week) {
    case "周一": System.out.println("干活"); break;
    case "周二": System.out.println("干活"); break;
    default: System.out.println("输入有误");
}
```

### for 循环执行顺序

```java
for (①初始化; ②条件; ④迭代) { ③循环体 }
// 执行顺序：① → ②(→③→④→②) → ②不成立退出
```

## 踩坑记录

> [!warning] switch 漏写 break 导致穿透
> **现象**：case 1 执行完后继续执行 case 2
> ```java
> int x = 1;
> switch(x) {
>     case 1: System.out.print("A");  // 没 break！
>     case 2: System.out.print("B"); break;
> }
> // 输出：AB（不是 A）
> ```

> [!warning] for 循环变量外部不可访问
> ```java
> for (int j = 0; j < 3; j++) { }
> System.out.println(j);  // ❌ 编译错误，j 不在作用域内
> ```
> 改成 while 可以解决：循环变量定义在外部。

> [!warning] do-while 容易忽略「至少执行一次」
> ```java
> do { System.out.println("执行了"); } while (false);
> // 虽然条件为 false，但「执行了」仍会打印一次
> ```

## 实战案例

```java
// 猜数字游戏
Random r = new Random();
int target = r.nextInt(100) + 1;  // [1, 100]
Scanner sc = new Scanner(System.in);
while (true) {
    int guess = sc.nextInt();
    if (guess > target) System.out.println("大了");
    else if (guess < target) System.out.println("小了");
    else { System.out.println("对了"); break; }
}
```
