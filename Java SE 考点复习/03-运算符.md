## 核心概念

### `+` 的两种身份

`+` 既是算术加号，又是字符串连接符。从左到右执行：

```
"abc" + 5       → "abc5"（拼接）
5 + 5           → 10（算术）
"abc" + 5 + 'a' → "abc5a"（字符串出现后全是拼接）
5 + 'a' + "xx" → "102xx"（先算 5+97=102，再拼接）
```

**关键判断**：字符串出现之前是算术运算，出现之后全是拼接。

### 短路运算符 `&&` 和 `||`

`&&` 左边为 false 右边**不执行**；`||` 左边为 true 右边**不执行**。

```java
int i = 10, j = 20;
System.out.println(i > 100 && ++j > 99);  // false
System.out.println(j);                     // 20（++j 没执行！）
```

## 语法格式

### 算术运算符

```java
int a = 10, b = 2;
a + b  → 12
a / b  → 5（整数除法）
5 / 2  → 2（不是 2.5，都是整数）
5.0 / 2 → 2.5（有浮点数参与就是浮点除法）
a % b  → 0（取余）
```

### 逻辑运算符

| 运算符 | 名称 | 例子 | 短路特性 |
|--------|------|------|---------|
| `&&` | 短路与 | `false && 任何` → false | ✅ 左侧 false 右侧不执行 |
| `\|\|` | 短路或 | `true \|\| 任何` → true | ✅ 左侧 true 右侧不执行 |
| `&` | 非短路与 | `false & ++j` → false | ❌ 两边都执行 |
| `\|` | 非短路或 | `true \| ++n` → true | ❌ 两边都执行 |
| `!` | 非 | `!true` → false | — |
| `^` | 异或 | `true ^ false` → true | 相同 false，不同 true |

## 踩坑记录

> [!warning] 字符串连接符陷阱
> **现象**：`"sum=" + 3 + 4` 结果是 `"sum=34"` 不是 `"sum=7"`
> **原因**：从左到右执行，遇到字符串后全部变拼接
> ```java
> System.out.println("sum=" + 3 + 4);    // "sum=34"
> System.out.println(3 + 4 + "=sum");    // "7=sum"（先算数）
> System.out.println("sum=" + (3 + 4));  // "sum=7"（先算括号）
> ```

> [!warning] 整数除法丢精度
> ```java
> double avg = (80 + 90 + 70) / 3;      // avg = 80.0 ❌
> double avg = (80 + 90 + 70) / 3.0;    // 80.0 ✅
> double avg = 1.0 * (80 + 90 + 70) / 3; // 80.0 ✅
> ```

> [!warning] 短路特性导致自增没执行
> ```java
> int x = 5, y = 5;
> boolean b = (x > 10) && (y++ > 0);
> // y = 5（不是 6！y++ 被短路了）
> ```
