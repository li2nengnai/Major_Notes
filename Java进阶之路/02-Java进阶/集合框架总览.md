# [[集合框架]]总览

## 核心概念

### 体系结构

```
Collection                    Map
├── List（有序可重复）         ├── HashMap
│   ├── ArrayList 数组        ├── LinkedHashMap
│   └── LinkedList 链表       ├── TreeMap
├── Set（不可重复）            ├── Hashtable（已过时）
│   ├── HashSet               └── ConcurrentHashMap（线程安全）
│   ├── LinkedHashSet
│   └── TreeSet
└── Queue（队列）
    ├── PriorityQueue
    └── ArrayDeque
```

### 快速选型

| 场景 | 选什么 |
|------|--------|
| 随机访问 | `ArrayList` |
| 频繁增删中间 | `LinkedList` |
| 键值对 | `HashMap` |
| 线程安全 Map | `ConcurrentHashMap` |
| 去重 | `HashSet` |
| 排序 | `TreeSet` / `TreeMap` |

### ArrayList 扩容

默认容量 10，扩容 1.5 倍：`newCapacity = old + (old >> 1)`

### LinkedList

双向链表，不扩容。随机访问 O(n)，头尾操作 O(1)。

## 踩坑记录

> [!warning] ArrayList 扩容开销大
> 如果事先知道数据量，指定初始容量：`new ArrayList<>(1000)`，减少扩容次数。

> [!warning] 迭代器遍历时不能调 list.remove()
> 会抛 `ConcurrentModificationException`，用迭代器的 `remove()`。
