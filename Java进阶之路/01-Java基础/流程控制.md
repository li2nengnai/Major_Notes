# [[流程控制]]

## 核心概念

### if-else vs switch

| 结构 | 适用场景 | 特点 |
|------|---------|------|
| `if-else` | 范围判断（成绩等级、年龄区间） | 灵活但嵌套多了可读性差 |
| `switch` | 等值匹配（星期、状态码） | 清晰，JDK14+ 支持表达式 |

```java
// if 做区间判断
if (score >= 90) grade = "A";
else if (score >= 80) grade = "B";

// switch 做等值匹配
// JDK14+ 箭头语法
String result = switch (day) {
    case 1, 2, 3, 4, 5 -> "工作日";
    case 6, 7 -> "休息日";
    default -> "非法值";
};
```

### 三种循环

| 循环 | 执行特点 | 适用场景 | 循环变量作用域 |
|------|---------|---------|---------------|
| `for` | 先判断后执行 | 知道循环次数 | 循环内部（外部不可访问） |
| `while` | 先判断后执行 | 不确定次数 | 外部（外部可访问） |
| `do-while` | **先执行一次**再判断 | 至少执行一次 | 外部 |

```java
// for —— 知道循环次数
for (int i = 0; i < 10; i++) { }
// System.out.println(i);  // ❌ 编译错误，i 不在作用域

// while —— 不确定次数
int i = 0;
while (i < 10) { i++; }
System.out.println(i);  // 10 ✅

// do-while —— 至少执行一次
do { System.out.println("执行了"); } while (false);
// 虽然条件为 false，但「执行了」会打印一次
```

**增强 for（for-each）**——遍历数组和集合的简便写法：

```java
int[] arr = {1, 2, 3};
for (int n : arr) {         // 不需要索引
    System.out.println(n);
}
```

### break vs continue（面试常问）

| 关键字 | 作用 | 影响范围 |
|--------|------|---------|
| `break` | 跳出当前整个循环 | 当前循环层 |
| `continue` | 跳过本次，进入下一次 | 本次迭代 |
| 带标签的 `break` | 跳出指定外层循环 | 指定标签层 |

```java
// 带标签的 break：跳出外层循环
outer:
for (int i = 0; i < 5; i++) {
    for (int j = 0; j < 5; j++) {
        if (j == 2) break outer;  // 直接跳出两层
    }
}
```

### 三元运算符

```java
int max = (a > b) ? a : b;
// 等价于：
int max;
if (a > b) max = a;
else max = b;
```

**不能嵌套太多层**，否则可读性差。一般只用于简单的二选一。

## 踩坑记录

> [!warning] switch 漏写 break 导致穿透
> **现象**：case 1 执行后继续执行了 case 2
> ```java
> int x = 1;
> switch(x) {
>     case 1: System.out.print("A");  // 没 break！
>     case 2: System.out.print("B"); break;
> }
> // 输出：AB（不是 A）
> ```
> **利用穿透**：故意不用 break 可以让多个 case 共享同一段代码：
> ```java
> switch (day) {
>     case 1: case 2: case 3: case 4: case 5:
>         System.out.println("工作日"); break;
>     case 6: case 7:
>         System.out.println("休息日"); break;
> }
> ```

> [!warning] do-while 至少执行一次的特性容易被忽视
> ```java
> do {
>     // 这里的代码不管条件成不成立都会执行一次
> } while (false);
> ```
> **建议**：能用 while 的地方不用 do-while，除非你真的需要「先做一次再说」。

> [!warning] for 循环变量在外部不可访问
> ```java
> for (int j = 0; j < 3; j++) { }
> System.out.println(j);  // ❌ 找不到符号
> ```
> **建议**：需要循环结束后用变量值时，用 while。
