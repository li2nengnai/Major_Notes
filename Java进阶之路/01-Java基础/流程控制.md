# [[流程控制]]

## 核心概念

### if-else vs switch

**if-else**：范围判断，灵活但代码可能冗长。  
**switch**：等值匹配，JDK7+ 支持 [[String]]，JDK14+ 支持箭头语法。

```java
// JDK14+ 箭头语法
String result = switch (day) {
    case 1, 2, 3, 4, 5 -> "工作日";
    case 6, 7 -> "休息日";
    default -> "非法值";
};
```

### 三种循环

| 循环 | 特点 |
|------|------|
| `for` | 知道次数，循环变量内部定义 |
| `while` | 不确定次数，先判断 |
| `do-while` | 至少执行一次 |

### break vs continue

- `break`：跳出当前循环
- `continue`：跳过本次，继续下一次
- 带标签的 `break/continue`：跳出多层循环

### 三元运算符

```java
int max = (a > b) ? a : b;
// 等价于 if (a > b) max = a; else max = b;
```
