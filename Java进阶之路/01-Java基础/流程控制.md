# Java 流程控制

> 来源：[二哥的Java进阶之路](https://javabetter.cn/basic-grammar/flow-control.html)

---

## 一、分支语句

### if-else
```java
if (条件1) {
    // 条件1成立
} else if (条件2) {
    // 条件1不成立，条件2成立
} else {
    // 都不成立
}
```

### switch（JDK 7+ 支持 String，JDK 14+ 支持表达式）
```java
// 传统写法
switch (day) {
    case 1:
    case 2:
    case 3:
    case 4:
    case 5:
        System.out.println("工作日");
        break;
    case 6:
    case 7:
        System.out.println("休息日");
        break;
    default:
        System.out.println("非法值");
}

// JDK 14+ 箭头语法 + switch 表达式
String result = switch (day) {
    case 1, 2, 3, 4, 5 -> "工作日";
    case 6, 7 -> "休息日";
    default -> "非法值";
};
// 配合 yield 返回值
String r = switch (day) {
    case 1: yield "周一";
    default: yield "其他";
};
```

## 二、循环语句

### for
```java
for (int i = 0; i < 10; i++) {
    System.out.println(i);
}

// 增强 for（for-each）
int[] arr = {1, 2, 3};
for (int n : arr) {
    System.out.println(n);
}
```

### while
```java
while (条件) {
    // 先判断后执行
}

do {
    // 至少执行一次
} while (条件);
```

### break vs continue
- `break`：跳出当前循环
- `continue`：跳过本次循环剩下的语句，进入下一次循环
- 带标签的 break/continue：跳出多层循环

## 三、三元运算符
```java
int max = (a > b) ? a : b;
// 等价于
int max;
if (a > b) max = a;
else max = b;
```
