# [[面向对象]]三大特性

## 核心概念

### [[封装]]

隐藏内部细节，通过 getter/setter 暴露访问接口。

```java
public class Person {
    private int age;
    public void setAge(int age) {
        if (age > 0 && age < 150) this.age = age;  // 校验
    }
}
```

**访问控制**：`private` → `default` → `protected` → `public`

### [[继承]]

子类 `extends` 父类，复用父类非私有成员。Java **单继承**（一个类只能有一个直接父类，可通过接口实现多继承）。

```java
public class Dog extends Animal {
    @Override
    public void eat() { /* 重写父类方法 */ }
}
```

**构造方法调用链**：父类静态代码块→子类静态代码块→父类构造器→子类构造器

### [[多态]]

同一个行为多种表现。三个条件：继承 + 重写 + 父类引用指向子类对象。

```java
Animal a = new Dog();
a.eat();  // 运行看右边（Dog 的 eat）
```

### [[抽象类]] vs [[接口]]

| 比较 | 抽象类 | 接口 |
|------|--------|------|
| 思想 | is-a 关系 | like-a 能力 |
| 构造器 | 有 | 无 |
| 多继承 | 不可以（单继承） | 可以（多实现） |

### 重写 vs 重载

| 对比 | 重写 Override | 重载 Overload |
|------|--------------|--------------|
| 范围 | 父子类之间 | 同类中 |
| 参数 | 相同 | 不同 |
| 返回值 | 相同或子类 | 无关 |

## 踩坑记录

> [!warning] 静态方法不能被重写
> 静态方法属于类，子类定义同名静态方法叫「方法隐藏」。调哪个看引用类型，不看实际对象。

> [!warning] 向上转型后不能调子类特有方法
> ```java
> Animal a = new Dog();
> a.bark();  // ❌ 编译错误
> ((Dog) a).bark();  // ✅ 向下转型
> ```
